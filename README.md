# Flask-API

1. Create a virtual environment and activate it

`$ mkdir myproject`
`$ cd myproject`
`$ python3 -m venv venv`
`$ . venv/bin/activate`


2. Install all dependencies from requirements.txt

`pip3 install -r requirements.txt `

3. Run the flask program 
`flask run`
4. A new database will be created from the CSV file :)
